Habilidad Code Morpher para OpenClaw.

## Qué Soluciona
Transforma bases de código heredadas en estructuras modernas y mantenibles.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque centrado en la seguridad
- Soporte para reversión

## Uso
/code-morpher

## Ejemplo
```
Analizar código en busca de problemas de codificación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
